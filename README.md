# Todo
It is a Todo App, i used Room, Coroutines, MVVM Architecuture, Material 3, live data and navigation components.

# Features:
* Create Tasks
* Read Tasks
* Check uncheck
* Update Tasks
* Delete Tasks

# Technologies used:
* Kotlin
* MVVM Architecture
* Manual Dependency injection
* Room
* Coroutines
* one Activity architecture
* Live data
* Views, XML
